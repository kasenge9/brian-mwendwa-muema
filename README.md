# Kinatwa 14-Seater Booking System

A lightweight front-end booking website for managing passenger reservations for a 14-seater shuttle.

## Features

- Capture customer details (name, phone, date, departure time).
- Visual seat map for all 14 seats.
- Prevent double-booking of seats.
- View current booking list.
- Persist bookings in browser `localStorage`.
- Clear all bookings with one click.

## Run locally

1. Clone this repository.
2. Open `index.html` in your browser.

No backend setup is required.
